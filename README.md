# R & Statistics Study Book

A free, comprehensive, interactive study guide for learning R programming and statistics for business analytics.

## 📚 Start Here

→ [Start Studying](https://nwfreidline.github.io/r-study-book) — Full interactive study book

## What Is This?

A self-contained HTML study resource covering foundational statistics and R programming — from loading data and fixing types through hypothesis testing and regression. Built from graduate-level coursework and designed for anyone learning R for data analysis.

No account required, no paywall, no ads. Just open it in a browser and start learning.

## What's Covered

| Module | Topic | Key Concepts |
|--------|-------|-------------|
| 1 | Introduction to R and RStudio | Setup, objects, vectors, data frames, data types, missing values |
| 2 | Descriptive Statistics | Central tendency, spread, skewness, kurtosis, histograms, boxplots |
| 3 | Probability Foundations | Probability rules, conditional probability, Bayes' theorem |
| 4 | Probability Distributions | Normal, binomial, Poisson distributions |
| 5 | Confidence Intervals | Estimation, margin of error, sample size |
| 6 | Hypothesis Testing | t-tests, ANOVA, chi-square, p-values |
| 7 | Correlation & Regression | Linear regression, correlation, model interpretation |

## Design

## How to Use

### Online (Recommended)
Visit [nwfreidline.github.io/r-study-book](https://nwfreidline.github.io/r-study-book)

### Offline
Download or clone this repo and open `index.html` in any browser.

```
index.html          ← Landing page with module links
module1.html        ← Module 1: Intro to R
module2.html        ← Module 2: Descriptive Statistics
...
```
